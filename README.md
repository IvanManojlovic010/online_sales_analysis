# Online Sales Analysis

##  Opis projekta
Ovaj projekat simulira sistem za upravljanje inventarom i korpom u online prodavnici.  
Korišćen je Python i koncepti objektno-orijentisanog programiranja (OOP).

##  Klase i funkcionalnosti

### Product
- Predstavlja jedan proizvod.
- Atributi: naziv, cena, količina.
- Metode: prikaz informacija o proizvodu.

### ProductManager
- Upravljanje listom proizvoda.
- Funkcionalnosti:
  - Dodavanje proizvoda
  - Uklanjanje proizvoda
  - Prikaz svih proizvoda
  - Računanje ukupne vrednosti inventara

### Cart
- Predstavlja korpu kupca.
- Funkcionalnosti:
  - Dodavanje proizvoda u korpu
  - Prikaz sadržaja korpe
  - Računanje ukupne vrednosti korpe
